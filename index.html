2) SEO: important “header/meta” tags to add

SEO basics for a page often includes a clear title, useful description, proper headings, and social preview metadata. Here is a practical list:

  • <title> — sets the page title shown in browser tab and search results.
  • <meta name="description"> — summary shown in search snippets.
  • <meta name="robots"> — indexing rules (e.g. index,follow).
  • <link rel="canonical"> — avoids duplicate-content issues by defining the “main” URL.
  • Heading tags: <h1>…<h6> — content hierarchy. Usually one h1 per page, then h2 for sections.
  • Open Graph tags (og:title / og:description) — helps link previews on social platforms.
  • <meta name="viewport"> — mobile friendliness (important for usability + ranking signals).

In this page, you can check the <head> section to see these tags included.

4b) Same text, 6 font sizes (inline styles; mix absolute + relative units)

Each list item explains when you would use that unit with a simple example.

  1. 12px (absolute): Use for small UI labels where you want exact control. Example: tiny captions under icons.
  2. 16pt (absolute, print-friendly): Use when designing for print/PDF-like layouts. Example: printable resume text size.
  3. 1rem (relative to root): Use for consistent sizing across the page. Example: base body text that adapts if the user changes default font size.
  4. 1.25em (relative to parent): Use for components that scale with their container. Example: a card title that grows if the card’s font size changes.
  5. 4vw (relative to viewport width): Use for responsive hero headings. Example: big landing-page header that scales on different screen widths.
  6. 120% (relative percentage): Use for slight emphasis relative to current size. Example: make a warning message 120% of the surrounding paragraph.
